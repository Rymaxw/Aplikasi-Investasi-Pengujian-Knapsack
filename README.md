analisis Perbandingan Kinerja Algoritma Knapsack Menggunakan Pendekatan Iteratif dan Rekursif pada Studi Kasus Optimisasi Portofolio Investasi Berbasis Web


Deskripsi:
Proyek ini berfokus pada analisis perbandingan efisiensi algoritma untuk menyelesaikan masalah optimisasi alokasi aset, sebuah tantangan umum dalam perencanaan keuangan terutama investasi. Sebuah web dibangun untuk mengimplementasikan solusi dari masalah klasik 0/1 Knapsack dan nantinya melalui pendekatan rekursif serta iteratif sebagai perbandingan performa, yang merepresentasikan skenario pemilihan aset investasi untuk memaksimalkan keuntungan dengan modal terbatas.
